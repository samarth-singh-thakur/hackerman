# GhostWatcher (MR. ROBOT themed Android app)

A Kotlin + Jetpack Compose Android app with a dark neon hacker aesthetic.


## Run
1. Open in Android Studio (Hedgehog+).
2. Let Gradle sync.
3. Run the `app` module.

> The app automatically detects if it's running on an emulator and adjusts the `ghostwatcher` API address to `http://10.0.2.2:8000` so it can reach your host machine.
